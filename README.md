# Fisiomove App

## Windows

### Install Python

1. Go to the official Python website:
   [https://www.python.org/downloads/windows/](https://www.python.org/downloads/windows/)
2. Download the **latest Python 3 release**
3. Run the installer
4. **IMPORTANT:** check the box **“Add Python to PATH”**
5. Click *Install*

To verify installation, open *Command Prompt* and run `python --version`, you should see something like `Python 3.12.x`.

### Run the application

Double‑click `run.bat`

## Linux (Debian/Ubuntu)

### Install Python

```
sudo apt update
sudo apt install -y python3 python3-venv
```

### Run the application

* `cd {project dir}`
* `./run.sh`
